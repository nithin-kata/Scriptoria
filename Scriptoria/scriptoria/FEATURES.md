# 🎬 Scriptoria - Complete Feature Documentation

## 🌟 Core Features

### 1. 📜 Screenplay Generation

**What it does:**
- Generates professional formatted screenplays from story concepts
- Includes proper scene headings (INT./EXT.)
- Character names in CAPS
- Action descriptions and dialogue
- Camera directions

**AI Model:** Mistral-7B-Instruct

**Output Format:**
```
INT. COFFEE SHOP - NIGHT

JOHN sits alone, staring at the rain outside.

JOHN
I never thought the night would 
change everything.
```

**Customization:**
- Adapts to genre (Action, Drama, Sci-Fi, etc.)
- Scales for Short Films or Feature Films
- Incorporates story concept details

---

### 2. 🎭 Character Profile Generation

**What it does:**
- Creates detailed character profiles
- Generates unique personalities
- Develops character motivations
- Designs emotional arcs

**Components:**
- **Name**: Character identity
- **Background**: History and context
- **Personality**: Key traits and behaviors
- **Motivation**: What drives the character
- **Emotional Arc**: Character transformation

**Display:**
- Beautiful card-based UI
- Hover animations
- Responsive grid layout
- Color-coded by importance

---

### 3. 🎬 AI Storyboard Generation

**What it does:**
- Generates visual representations of key scenes
- Creates cinematic images using Stable Diffusion XL
- Provides camera angle suggestions
- Includes lighting recommendations

**AI Model:** Stable Diffusion XL Base 1.0

**Scenes Generated:**
1. Opening Scene
2. Mid-Point Tension
3. Climactic Finale

**For Each Scene:**
- AI-generated image
- Camera angle (Wide, Close-up, etc.)
- Lighting setup (Natural, Dramatic, etc.)
- Scene description

**Image Quality:**
- High resolution
- Cinematic composition
- Genre-appropriate styling
- Professional film aesthetics

---

### 4. 🎧 Sound Design Suggestions

**What it does:**
- Creates comprehensive audio landscape
- Suggests music styles
- Recommends ambient sounds
- Defines emotional audio cues

**Components:**

**Music:**
- Genre-appropriate styles
- Instrumentation suggestions
- Tempo and mood

**Ambient Sounds:**
- Environmental audio
- Background atmosphere
- Spatial audio cues

**Emotional Cues:**

- Dramatic emphasis

**Example Output:**

```
Emotional: Building mystery and anticipation
```

---

### 5. 💰 Production Budget Estimation

**What it does:**
- Calculates comprehensive production costs
- Breaks down by department
- Scales based on film duration
- Adjusts for cast size

**Budget Categories:**

1. **Cast** - Actor salaries and talent fees
2. **Crew** - Production team costs
3. **Equipment** - Camera, lighting, grip
4. **Locations** - Venue rentals and permits
5. **Sound** - Audio equipment and post
6. **Post-Production** - Editing and VFX

**Calculation Factors:**
- Film duration (Short vs Feature)
- Number of characters
- Genre complexity
- Production scale

**Display:**
- Clean metric cards
- Visual breakdown
- Total budget highlight
- Professional formatting
